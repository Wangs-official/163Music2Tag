# 163Music2Tag

拒绝瞎几把收费，享受美好本地人生

> CAONIMA , API被Vercel搞了，先用别的服务吧

## 使用

1. 安装

`python install.py`

2. 使用

`python main.py -id {歌曲ID}`

3. 配置设置文件（可忽略）

```yaml
api_url: https://m163.a.vercel.stardawn.xyz/ # API域名
del_tmp_when_complete: false # 是否删除缓存在每次完成之后
use_cookie: false # 是否使用Cookie请求
```
